# Customer Address Book

## Project Setup

clone the Repository

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

## Used Technologies 

◽ Vue.js 3.3

◽ JavaScript

◽ Tailwindcss

◽ Vuetify
